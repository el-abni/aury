# CHANGELOG

All notable changes to **💜 Aury** are documented in this file.

---

## 1.4.0

### Added
- Improved natural language understanding for terminal commands
- Support for helper words like `quero`, `pode`, `por favor`
- Support for vocative expressions like `Aury,`
- Support for punctuation in commands
- File command connectors such as `para` and `em`

### Improved
- Interpretation of multiple intentions in a single command
- Interpretation of multiple targets (ex: `ver cpu e memória`)
- File command argument parsing
- Overall natural language parsing reliability

### Examples

aury quero instalar obs studio
aury ver cpu e memória
aury atualiza, otimiza e baixa o firefox
aury mover arquivo teste.txt para pasta/teste.txt
